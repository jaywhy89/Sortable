# Sortable Challenge

This program matches product listings against a set of known products.

## Getting Started

### Installing

```
git clone https://github.com/jaywhy89/Sortable.git
```
or click [here](https://github.com/jaywhy89/Sortable)

## Running the tests
After cloning,
```
cd Sortable
```

Type blow to compile
```
make
```
Type below to run
```
make run
```
Type below to clean
```
make clean
```
Open "results.txt" to check output
